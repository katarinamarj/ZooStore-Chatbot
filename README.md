This application allows users to browse a selection of pets and add their chosen pets to the shopping cart. The app provides a simple and intuitive interface for exploring available pets.

Additionally, the application includes a chatbot feature, allowing users to ask questions and receive assistance directly within the app. The frontend is built using Angular, while the chatbot is powered by Rasa, enabling interactive and intelligent user communication.
